# 🧠 Quiz Application (Python)

## 📌 Overview

This project is a **command-line Quiz Application** developed using Python.
It allows users to attempt multiple-choice questions and calculates the final score based on correct answers.

This project is designed to demonstrate **core Python concepts**, clean code structure, and basic application logic.

---

## 🚀 Features

* 📋 Multiple-choice questions
* ✅ Instant answer validation
* 📊 Score calculation
* 🧩 Modular code structure
* 👨‍💻 Beginner-friendly and easy to understand

---

## 🛠️ Tech Stack

* **Language:** Python
* **Concepts Used:** Functions, Lists, Conditionals, Loops

---

## 📂 Project Structure

```
Quiz/
│── main.py          # Entry point of the application
│── questions.py     # Contains all quiz questions
│── README.md        # Project documentation
```

---

## ▶️ How to Run Locally

1. Make sure Python is installed on your system
2. Clone the repository:

   ```
   git clone https://github.com/Ayush9087/Quiz-.git
   ```
3. Navigate to the project directory:

   ```
   cd Quiz-
   ```
4. Run the application:

   ```
   python main.py
   ```
